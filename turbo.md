# turbo 配置

## `"ui": "tui"`的意思是

- TUI 是 Terminal User Interface（终端用户界面）的缩写。
- 它的主要作用：
  > 当你运行 turbo dev 或 turbo run 同时启动多个微服务时，传统的行为是把所有服务的日志全部交织在一起打在屏幕上，非常混乱。开启 tui 后，Turborepo 会提供一个交互式的可视化界面：

1. 任务列表：在终端左侧或顶部显示所有正在运行的任务及其状态（运行中、成功、失败）。
2. 专注日志：你可以使用键盘（通常是上下键）切换不同的服务，只查看当前选中服务的日志，而不会被其他服务的日志干扰。
3. 实时状态：一眼就能看出哪个服务崩溃了。
